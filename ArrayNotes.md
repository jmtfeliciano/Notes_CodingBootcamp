# Dealing with arrays

Just general notes to remember about arrays

## Manipulating ends of array

### Beginning of array 

```
var array1 = [1,2,3,4];
array1.shift(); 
//console.log(array1) will print [2,3,4]

var array2 = ['a','b','c','d'];
array2.unshift('1'); 
//console.log(array2) will print ['1','a','b','c','d'];
```
