# Basics of Ruby

## Basic difference between puts and print

puts behaves like print in java.
and print behaves like println in java or equivalent to javascript's print

```ruby
print "he"
puts "hehe" 
//console will print hehe

```

## Ruby has a .reverse method too!

```ruby
"hello".reverse
//will return olleh

```

## upcase and downcase

```ruby
puts "hello".upcase 
// prints HELLO
puts "Hi".downcase
// prints hi
```
